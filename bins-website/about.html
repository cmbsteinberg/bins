About This Service

Find bin collection schedules across 300+ UK councils in one place

What We Do

UK Bin Lookup is a free service that helps you quickly find your bin collection schedule. We've integrated with over 300 UK local councils to provide a unified lookup experience. No more searching through council websites or downloading PDF calendars.

How It Works

  1. Enter Your Postcode

    We use your postcode to find all addresses in your area and identify your local council.

  2. Select Your Address

    We look up your UPRN (Unique Property Reference Number) - a unique identifier for your property.

  3. Fetch Collection Data

    We query your council's API directly to get your bin collection schedule in real-time.

  4. View Your Schedule

    See your upcoming bin collections with dates and bin types (recycling, general waste, garden waste, etc.).

Supported Councils

We currently support 306 UK councils with API-based collection lookups, including:

  • 185 councils with single API endpoint lookup
  • 79 councils with ID-based lookup systems
  • 42 councils with token-based authentication

Coverage includes councils across England, Scotland, Wales, and Northern Ireland.

Privacy & Data

This service runs entirely in your browser. We don't store your postcode, address, or any personal information. All data is fetched directly from official council APIs and government services.

We use:

  • GOV.UK API - For council identification
  • Council APIs - For bin collection schedules
  • UPRN Lookup - For address resolution

Technology

This is a simple, fast, privacy-focused web application built with:

  • Plain HTML, CSS, and JavaScript (no frameworks)
  • Pico CSS for beautiful, accessible styling
  • Cloudflare Pages for hosting
  • Cloudflare Workers for CORS proxy

Limitations

While we support 300+ councils, some limitations apply:

  • We only support councils with API-based lookup (no browser automation required)
  • Some councils may have temporary outages or API changes
  • Collection dates depend on the accuracy of council data
  • Special collections (e.g., Christmas schedules) may not be reflected immediately

Open Source

This project builds on council configuration data extracted and maintained as part of an open-source effort to make UK bin collection data more accessible. The underlying Python library and council configurations are available for developers and researchers.
